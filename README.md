# 贪吃蛇 🐍

用 HTML + CSS + JavaScript 写的经典贪吃蛇小游戏。

## 玩法

- **方向键** 或 **WASD** 控制蛇的移动
- **空格** 暂停 / 继续
- 吃到红色食物得分，随分数提升等级，蛇会移动得更快
- 撞墙或撞到自己则游戏结束

## 本地运行

直接用浏览器打开 `index.html` 即可，或使用任意静态服务器，例如：

```bash
# 若已安装 Python 3
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 部署到 GitHub Pages

1. 把本仓库 push 到 GitHub
2. 仓库设置 → Pages → Source 选择 `main` 分支（或你使用的默认分支）
3. 保存后访问 `https://<你的用户名>.github.io/<仓库名>/`

Enjoy!
